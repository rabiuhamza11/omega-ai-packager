# OMEGA AI Packager (omega-pack) v0.3.0

Ship AI agent projects like software: package, deploy, version, observe.

## Overview
omega-pack is the official CLI and SDK for building, packaging, and deploying AI agent projects in the OMEGA INFINITY 1000 ecosystem. Includes a full RAG engine, multi-user workspace management, and enterprise observability.

## Features
- RAG knowledge engine (TF-IDF scoring across indexed chunks)
- Multi-user workspace management
- Semantic search across indexed documentation
- Sentry + Langfuse observability integration
- Plugin-ready architecture (UPEMF v25.0 compliant)
